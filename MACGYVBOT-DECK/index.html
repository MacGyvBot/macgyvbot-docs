Perception / Hand-eye Calibration

Hand-eye calibration 절차와 좌표 변환 검증

CALIBRATION TARGET

RGB-D camera frame
→ robot base frame

YOLO/SAM/depth로 얻은 공구 픽셀과 depth를 camera 좌표로 복원하고, hand-eye transform을 적용해 로봇 base 기준 pick pose로 변환합니다.

Tbase←campbase = Tbase←cam · K-1[u, v, 1]d
PROCEDURE

고정 checkerboard를 로봇 자세별로 촬영하고, 각 이미지와 현재 gripper pose를 함께 저장했습니다.

SOLVE

체커보드 코너 검출, solvePnP, cv2.calibrateHandEye를 실행해
gripper → camera 변환을 계산했습니다.

OUTPUT

결과는 4x4 변환 행렬로 저장하고, 실행 시
base → gripper @ gripper → camera 조합으로 camera 좌표를 base 좌표로 변환했습니다.
